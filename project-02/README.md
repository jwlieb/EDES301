#EDES301 Project 2: Talking Fish PCB

This project is a compact printed circuit board for the talking fish assistant implementation explored in Project 1.

The EAGLE folder contains the library, schematic, and layout for the PCB. 

The MFG folder contains the manufacturing information including a BOM, gerber files, and additional screenshots for assembly.

More information can be found in the docs folder, which houses the system block diagram, mechanical drawing, schematics pdf, and addional images of the top and bottom layers of the board.
